# Draft-to-Funding Markets — Demo

Interactive animated explainer for **BetFundMe** and Draft-to-Funding Markets.

**[→ View Demo](https://kindainsecurebot.github.io/betfundme-demo/)**

## What is this?

Bets on funding outcomes can be *drafted* — partially converted into actual funding — to justify reaching thresholds. The protocol accounts as if coordination happened, because mathematically it should have.

## Run locally

Open `index.html` in a browser. That's it — single self-contained file, no build step.

## Controls

- **Click / →** — advance
- **← / Backspace** — go back
- **Swipe** — mobile navigation
